UI / UX CASE STUDY

Bakora

Personal Finance Mobile Application

Bakora is a mobile finance application designed to help users manage their income, expenses, savings and financial goals through a simple, modern and user-friendly interface.

Project Overview

Bakora was designed to simplify personal finance management by giving users one place to monitor their money, analyse spending, save towards goals and complete transfers effortlessly. The focus of the project was creating a clean, intuitive and visually engaging mobile experience.

Role

UI/UX Designer

Timeline

4 Weeks

Tools

Figma

Platform

Mobile App

The Problem

Managing finances can become overwhelming when users must switch between multiple applications to monitor transactions, savings, expenses and transfers. Existing solutions are often cluttered, making it difficult for users to quickly understand their financial position.

The Solution

Bakora provides a clean financial dashboard where users can manage expenses, complete transfers, monitor savings progress and stay informed through an intuitive mobile experience.

My Role

Research

Conducted user research to understand user needs, financial pain points and opportunities for improvement.

Wireframing

Created low-fidelity wireframes to define the application structure and user flow before designing high-fidelity screens.

UI Design

Designed modern mobile interfaces with a focus on simplicity, accessibility and visual consistency.

Prototyping

Built interactive prototypes in Figma to validate navigation and overall user experience.

Key Features

Dashboard

A clear financial overview showing balances, spending and recent activity.

Expense Tracking

Easily monitor spending habits across different categories.

Savings Goals

Track progress towards financial goals with visual progress indicators.

Money Transfer

A simple transfer experience designed for speed, clarity and ease of use.

Project Reflection

Designing Bakora strengthened my understanding of user-centred design, information architecture, mobile-first thinking and financial product design. The project improved my ability to translate research into practical interface solutions while creating visually appealing and intuitive user experiences.

Frontend Development Journey

Although Bakora began as a UI/UX project, I am currently recreating parts of the design using HTML, CSS and JavaScript to strengthen my frontend development skills and gain a deeper understanding of responsive web implementation.
